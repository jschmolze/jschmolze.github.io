Time-Inhomogeneous Markov Chains for JPM

Python • Deep Learning • Stochastic Processes

Research project modeling JPMorgan Chase daily return dynamics using a time-dependent Markov chain learned via neural networks. Discretizes returns into states and trains an MLP to estimate conditional transition matrices given current state and macro/fundamental signals, enabling probabilistic forecasting, regime analysis, and risk simulation.
